# FitPath - Fitness Tracking App 🏋️ 🏋️

FitPath is a fitness tracking mobile application built using React Native, Expo, TypeScript, and Supabase.

## Features

- User Authentication
- Workout Tracking
- Exercise Library
- Progress Monitoring
- Admin Dashboard
- Offline Support
- Push Notifications
- Profile Management

## Tech Stack

- React Native
- Expo Router
- TypeScript
- Supabase
- NativeWind (Tailwind CSS)

## Project Overview

FitPath helps users track workouts, monitor fitness progress, and stay consistent with their fitness journey through a modern mobile application.

## Key Functionalities

- Create and manage workouts
- Track workout history
- Monitor fitness progress
- User authentication and profiles
- Admin controls
- Offline data support
- Exercise database

## Future Improvements

- AI Workout Recommendations
- Nutrition Tracking
- Social Features
- Advanced Analytics
- Wearable Device Integration
